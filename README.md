# GYMPROJECT 🏋️‍♂️

This is an Object-Oriented Programming (OOP) based gym management system developed in Java. It models key components of a gym such as trainers, programs, management, and payments.

## 🚀 Features

- 💪 Fitness, Pilates, Combat, and Swimming Programs
- 👨‍🏫 Different types of Trainers with polymorphism
- 📅 Payment and Salary Management
- 🧼 Cleaning tasks and Diet plans
- 🔄 Object-oriented design with inheritance and encapsulation

## 📁 Project Structure

```
gym/
 ├── idea/
 │   ├── management/     → Payments, Salaries, Plans
 │   ├── trainers/       → Trainer classes (Fitness, Pilates, etc.)
 │   └── programs/       → Gym programs (Swimming, Diet, etc.)
```

## 🛠️ Technologies

- Java
- IntelliJ IDEA
- Object-Oriented Programming principles

## ▶️ How to Run

1. Open the project in IntelliJ IDEA
2. Locate `Main.class` or `Main.java` (if available)
3. Run the `Main` file to start the program

## 📌 Notes

- This project is a sample educational application for OOP concepts.
- Make sure the correct classpaths are set if running manually.
